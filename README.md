# 🎸 Chord Detection using Deep Learning

## 📌 Description
A deep learning-based system for chord recognition from audio files using MFCC features and CNN. The model processes `.wav` audio input and predicts musical chords accurately.

---

## 🚀 Features
- Audio chord prediction from `.wav` files
- MFCC feature extraction using Librosa
- CNN-based classification using TensorFlow/Keras
- Supports real-time file input

---

## 🛠️ Technologies Used
- Python
- NumPy
- Librosa
- TensorFlow / Keras
- Scikit-learn
